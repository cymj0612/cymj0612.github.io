Hexo与GitHub搭建个人博客

Posted on 2018-06-26

#一、 概述

1.Hexo

Hexo 是一个快速、简洁且高效的静态博客框架。

2.Git与Github

Git是一种版本控制系统，Github是一个网站，给用户提供git服务，类似于百度云盘储存文件。

3.相关软件

1.Git

2.Node.js基于 Chrome V8 引擎的 JavaScript 运行环境。

3.Sublime Text编译器

#二、Github准备

在[Github官网](https://github.com/)注册账号（建议使用谷歌浏览器，自带翻译功能），新建项目，项目名称：

**Github账号名.github.io**

#三、搭建工作

1.检验Node是否安装。win+R 下进入 cmd 命令行窗口，输入命令：

node -v

2.安装Hexo ，在 cmd 命令行窗口输入命令：

npm install hexo-cli -g

3.在合适的盘中新建文件夹，右键点击 Git Bash Here,依次输入如下命令：

git config --global user.name "Github账户名称"
git config --global user.email"邮箱"
hexo init
hexo install
npm install hexo-deployer-git --save

4. 在 Github之前新建项目下，找到项目地址并复制：

https://github.com/Github账号名/Github账号名.github.io.git

5.打开Sublime Text3编译器，将新建的博客文件夹鼠标拖入Sublime主页，打开左侧目录栏中_config.yml文件，修改文末deployment如下：

deploy：
  type: git（注意冒号后有空格，正确颜色为：type红色，git黄色）
  repo: https://github.com/cymj0612/cymj0612.github.io.git
  branch: master
一定要先保存！！！

6.在Git 窗口运行命令：

hexo g -d

7.检验Github新建项目中是否上传文件，然后复制该网页网址末端并搜索：账户名称.github.io，即可进入个人博客。

云水禅心

Posted on 2018-06-25 
曾多次提笔，想要续写这样一段风细柳斜的心事，却在回眸的刹那，忘却了这世间的繁华。我不知道，在何时，才能够放下云水过往里所有不见不散的谎言，背上禅的行囊，坐看云起时。

有人说，背上行囊，就是过客，放下包袱，便是归人。只是俗世的你我，有几人可以放下繁华的世相，出离尘世，拈花笑佛，煮茶听雨，将万丈红尘过到世事忘记，绿云无扰。

烟雨红尘，步步生莲。我想此生，做个归人，不是过客，不再云海漂泊，离群孤索。

这世间，有太多，不属于我的繁华。红尘万象，给的了我喧嚣热闹，却无法掩藏背后的孤寂与苍凉。
